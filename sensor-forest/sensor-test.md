# Sensor Test

REGLAS
1. Si hay movimiento y si hay sonido = actividad muy sospechosa
2. Si hay movimiento y no hay sonido = actividad sospechosa
3. Si no hay movimiento y no hay sonido = actividad segura

CASOS:

1. Se metieron a una tienda a robar y hay mucho ruido = actividad muy sospechosa
2. Una persona entró a la tienda cuando ya estaba cerrada = actividad sospechosa
3. Al cerrar la tienda todo permanece como estaba = actividad segura
